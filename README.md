# Win95 Investor MVP

Win95-style investor critique MVP with a static frontend and a Node.js backend.

## Requirements

- Node.js 18+

## Run Locally

```bash
npm start
```

The app runs at `http://localhost:4173` by default. Set `PORT` to use a different port.

## Features

- Upload or paste a business plan
- Select an investor-style critique lens
- Generate feasibility scores, red flags, MVP logic, and due diligence questions
